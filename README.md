## FukuAPI

A modern REST API with a sleek, animated, responsive frontend.

### Setup
1. Install dependencies: `npm install`
2. Run locally: `npm start`
3. Visit: http://localhost:3000

### Deployment
Deploy to Vercel using the vercel.json config.
